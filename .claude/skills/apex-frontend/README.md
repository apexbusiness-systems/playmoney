# apex-frontend (Claude Skill)

Install in Claude.ai: Settings → Capabilities → Skills → Upload → select the ZIP.

Invoke:
- /apex-frontend design <goal>
- /apex-frontend audit <app or screenshots>
- /apex-frontend implement <design/spec>
- /apex-frontend debug <bug + repro>
- /apex-frontend perf <scenario>
- /apex-frontend a11y <screens>
- /apex-frontend system <design system needs>
- /apex-frontend migrate <from> <to>

If you already have an older `apex-frontend` installed, remove it first and re-upload to avoid conflicts.
