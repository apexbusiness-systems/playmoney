# Audit Playbook

## Failures (❌)
- Visual redesign without funnel diagnosis.
- Opinion-driven changes with no measurement.

## Correct Pattern (✅)
Evidence → Friction map → Hypotheses → Prioritize (ICE) → Ship smallest fixes → Measure.

## Friction Map Template
Step | Intent | Friction | Severity | Hypothesis | Fix | Metric.

## Severity
P0 blocks task; P1 heavy friction; P2 annoyance; P3 polish.
