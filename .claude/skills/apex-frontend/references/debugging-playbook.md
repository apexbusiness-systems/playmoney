# Debugging Playbook

## Failures (❌)
- No repro → guessing.
- Fixing symptoms → recurring bug.

## Deterministic protocol (✅)
Reproduce → Reduce → Classify → Instrument → Inspect → Patch → Prevent.

## Classification
Layout/constraints | state/race | render/perf | data/API | platform | build/tooling.

## Prevention
Regression test + guardrail + monitor event.
