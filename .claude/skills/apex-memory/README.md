## Installation

### For Claude

1. Download `apex-memory-v2-claude.zip`
2. Go to Claude.ai → Settings → Skills
3. Upload the zip file
4. APEX-Memory activates automatically

### For Any AI Agent

```bash
# Extract and copy to your agent's skills directory
unzip apex-memory-v2-claude.zip -d /path/to/your/agent/skills/apex-memory
```
